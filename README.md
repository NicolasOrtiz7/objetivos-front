# AppObjetivos

Front end: Angular
Back end: Spring boot
Base de datos: MySQL

## Características
- Cards personalizadas de objetivos
- Crear objetivos con nombre, descripción, imagen, tiempo y dinero límite, poder finalizar y reanudar
- Barras de progreso de tiempo y dinero
